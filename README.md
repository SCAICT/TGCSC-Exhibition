# TGCSC-Exhibition

## Project Tree

- GameConsole
    - 與會眾互動的程式，接收來自CardSensor的Serial資料
    - C# Windows Form
- queryServer
    - 處理GameConsole之SQL查詢請求，並回傳資料
    - Python Flask
- CardSensor
    - 實體裝置電阻測量，藉此分辨SQL Syntax卡片
    - Clang Arduino
- H-Bride-7A-Motor-CTR
    - 控制馬達風扇
    - 採3D列印裝置，Clang Arduino
